Burti - Terms of Service

Last updated: March 3, 2026

1. Service Description

Burti is a personal content automation tool that generates and publishes original webcomic episodes featuring the character "Burti" across social media platforms including TikTok and Instagram.

2. Usage

This application is operated solely by the creator of the Burti character for personal content publishing. It is not a public-facing service and does not accept third-party users.

3. Content

All content generated and published through this tool is original creative work. The tool posts content only to accounts owned and operated by the Burti creator.

4. API Usage

This application accesses the TikTok Content Posting API solely to upload original video content to the creator's own TikTok account (@burti.kim). No user data from other TikTok users is collected or processed.

For questions, contact: burti.kim (at) TikTok / Instagram
